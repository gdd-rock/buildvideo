# 参考图转角色设定图提示词（图生图模式）

## 核心要求：高度还原参考图人物

**必须严格还原参考图中人物的所有视觉特征**，包括但不限于：
- 面部五官（眼型、鼻型、嘴型、脸型轮廓）必须与参考图高度一致
- 发型（长度、颜色、造型、刘海）必须完全一致
- 体型和身材比例必须一致
- 服装款式、颜色、纹理、配饰必须完全一致
- 肤色、肤质必须一致

生成的角色必须让人一眼就能认出是参考图中的同一个人物。

## 画风保持规则

保持原图的画风和艺术风格：
- 真人照片风格 → 生成真人风格
- 动漫风格 → 生成动漫风格
- 写实插画 → 生成写实插画风格
- 其他风格 → 保持一致

## 生成规则

1. 忽略原图的具体色调和光线
2. 使用自然柔和的摄影棚灯光
3. 绘制正常美观的人体比例
4. 不要复制原图的画质、模糊、噪点或瑕疵
5. 生成的图像必须清晰锐利、细节丰富、专业品质
6. 角色表情应为自然平静的中性表情，目光正视镜头

## 缺失部位自动补齐

如果参考图是半身或部分身体，请根据服装风格和人物特征合理补全未露出的部位：
- **缺少下半身**：根据上衣风格推断并绘制匹配的裤装/裙装
- **缺少脚部**：根据整体穿搭风格添加合适的鞋款
- **缺少手部/手臂**：根据姿态合理补全
- 保持整体风格一致，确保补全的部分与可见部分协调统一
